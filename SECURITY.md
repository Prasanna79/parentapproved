# Security Policy

## Supported Versions

We currently support and provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 0.9.x   | :white_check_mark: |
| < 0.9.0 | :x:                |

## Reporting a Vulnerability

We take the security of ParentApproved.tv seriously. If you believe you have found a security vulnerability, please report it to us by emailing:

**hello@parentapproved.tv**

Please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact if exploited.

We will acknowledge receipt of your report within 48 hours and provide a timeline for resolution.

### Privacy First

As ParentApproved.tv is an app for kids, we prioritize privacy and security above all else. We welcome responsible disclosure from security researchers and the community.
